# Notes for Android

A simple note-taking Android app, built with Jetpack Compose.

### Authenticating

```
http POST http://localhost:3000/api/users \
  email=user2137@gmail.com \
  password=Tiger123
```

```
http POST http://localhost:3000/api/auth/login \
  email=user2137@gmail.com \
  password=Tiger123
```
